Project 3: LTI System Response to Random Input

2025, Rayhan R. Zamzamy, et al.

In this experiment, we evaluate how a linear time-invariant (LTI) system responds to a random input signal. The system is implemented in Moving Average (MA) filters. We simulate the LTI system and measure the characteristics of its output signal, including mean, power, autocorrelation, and power spectral density (PSD). This experiment demonstrates the frequency-selective and memory-related effects of LTI systems on stochastic input signals.

Preparation

In[1]:
# Set the seed for reproducibility
set.seed(52)

1. Generate a Random Input Signal

We simulate a white Gaussian noise signal as the random input to the system.

In[2]:
n <- 1000
x <- rnorm(n, mean = 0, sd = 1)

# Plot the white noise signal
plot.ts(x, main = "Input Signal (White Noise)", ylab = "x(t)")

2. Define the LTI System

We define the LTI system as a Moving Average (MA) filter with the following transfer function:

$$ H(z) = \frac{1}{3}(1 + z^{-1} + z^{-2}) $$

This is a simple FIR low-pass filter. We will visualize:

  • Magnitude response
  • Phase response
In[3]:
# Define LTI filter coefficients: MA with equal weights
b <- rep(1/3, 3)

# Frequency response of the system
freq_response <- fft(c(b, rep(0, n - length(b))))
freq_bins <- seq(0, 0.5, length.out = 501)

# Plot frequency response
par(mfrow = c(1, 2))
options(repr.plot.width = 10, repr.plot.height = 5)

plot(freq_bins, Mod(freq_response[1:501]), type = "l", col = "blue", lwd = 2,
     main = "Magnitude Response of MA Filter", 
     xlab = "Normalized Frequency", ylab = "Magnitude")
plot(freq_bins, Arg(freq_response[1:501]), type = "l", col = "darkred", lwd = 2,
     main = "Phase Response of MA Filter", 
     xlab = "Normalized Frequency", ylab = "Phase (radians)")

3. Simulate LTI System with the Random Input

We apply the MA filter to the white noise input signal.

In[4]:
# Simulate LTI system output
y <- filter(x, filter = b, sides = 1)

# Plot the output signal
plot.ts(y, main = "Output Signal y[n]  Response of MA Filter", ylab = "Y[n]")

# Save to a file
write.csv(data.frame(input = x, output = y), "ma_filter_output.csv", row.names = FALSE)

4. Analyze Data

In[5]:
y_valid <- y[!is.na(y)]
x_valid <- x[1:length(y_valid)]

# Output statistics (simulated & theoretical)
mean_y <- mean(y_valid)
power_y <- mean(y_valid^2)
mean_y_theo <- 0
power_y_theo <- sum(b^2)

# ACF (simulated & theoretical)
acf_y <- acf(y_valid, lag.max = 10, plot = FALSE)$acf[,,1]

acf_y_theo <- numeric(11)
for (k in 0:10) {
  if (k <= (length(b) - 1)) {
    acf_y_theo[k + 1] <- sum(b[1:(length(b) - k)] * b[(k + 1):length(b)])
  } else {
    acf_y_theo[k + 1] <- 0
  }
}

# PSD (simulated & theoretical)
spec_y <- spectrum(y_valid, plot = FALSE)

omega <- seq(0, pi, length.out = 512)
H <- b[1]*exp(-1i*omega*0) + b[2]*exp(-1i*omega*1) + b[3]*exp(-1i*omega*2)
psd_y_theo <- Mod(H)^2  # since Sxx = 1

# Tables
summary_y <- data.frame(
  Quantity = c("Mean", "Power"),
  Simulated = c(mean_y, power_y),
  Theoretical = c(mean_y_theo, power_y_theo)
)

acf_table_y <- data.frame(
  Lag = 0:3,
  ACF_Simulated = round(acf_y[1:4], 4),
  ACF_Theoretical = round(acf_y_theo[1:4], 4)
)

psd_table_y <- data.frame(
  Freq = round(spec_y$freq[1:5], 4),
  PSD_Sim = round(spec_y$spec[1:5], 4),
  PSD_Theo = round(psd_y_theo[1:5], 4)
)

# Plots
par(mfrow = c(2, 2))
options(repr.plot.width = 15, repr.plot.height = 10)

plot.ts(x, main = "Input White Noise", ylab = "x[n]")
plot.ts(y, main = "Output of MA Filter", ylab = "y[n]")
acf(y_valid, lag.max = 10, main = "ACF of Output y[n]")
plot(spec_y$freq, spec_y$spec, type = "l", col = "blue",
     main = "PSD of Output y[n]", xlab = "Frequency", ylab = "Power")
lines(omega/(2*pi), psd_y_theo, col = "red", lty = 2)
legend("topright", legend = c("Simulated", "Theoretical"), col = c("blue", "red"), lty = 1:2)
